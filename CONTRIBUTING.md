# 🤝 Contributing to Agozilla

Thank you for your interest in contributing!

## Getting Started

Clone the repository

```bash
git clone https://github.com/Prathvikmehra/agozilla.git
```

Install dependencies

```bash
npm install
```

Run the test suite

```bash
npm test
```

## Development

Please keep pull requests focused on a single feature or bug fix.

Before submitting:

- Run all tests
- Update documentation if needed
- Follow the existing code style

## Reporting Bugs

If you find a bug, please open an issue with:

- Node.js version
- Steps to reproduce
- Expected behavior
- Actual behavior

## Feature Requests

Feature requests are welcome!

Please explain:

- Why the feature is useful
- Example use cases

Thank you for helping improve Agozilla ❤️
